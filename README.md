# RegularPolyhedron
YMM4用の正多面体描画プラグイン <br> 
 
 
# インストール方法
Releaseにある最新バージョンのymmeファイルをダウンロードし、ダブルクリックでインストール。  <br>
 
 
# 機能
正多面体を描画し、面に画像を貼り付けたり、回転させるなど。  <br>
 
 
# 使い方
YMM4でタイムライン上に図形を追加し、図形の種類を「正多面体」に変更。  <br>
好きにプロパティを編集する。  <br>

# 動作要件
OS: Windows 10 / 11 (64bit)  <br>
YukkuriMovieMaker4: 最新版を推奨  <br>
ランタイム: .NET 10.0  <br>
